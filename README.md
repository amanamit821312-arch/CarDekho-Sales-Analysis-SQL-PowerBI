# CarDekho-Sales-Analysis-SQL-PowerBI
Analyze CarDekho vehicle data using SQL and Power BI to uncover pricing trends, fuel preferences, transmission distribution, seller insights, and vehicle depreciation patterns through interactive dashboards.

## 📌 Project Overview

This project analyzes the CarDekho vehicle dataset using SQL and Power BI to uncover key insights into the used car market. The analysis focuses on vehicle pricing, fuel preferences, transmission trends, seller types, ownership patterns, and depreciation behavior.

The goal is to transform raw vehicle data into meaningful business insights through data exploration, SQL queries, and interactive Power BI dashboards.

## 🎯 Objectives

* Analyze vehicle pricing trends.
* Compare manual and automatic transmission popularity.
* Identify the most preferred fuel types.
* Evaluate seller type distribution.
* Study vehicle depreciation patterns.
* Generate actionable business insights using visual analytics.

## 🗂 Dataset Information

The dataset contains information about:

* Car Name
* Year
* Selling Price
* Present Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission Type
* Number of Previous Owners

## 📈 Key Insights

* Manual transmission vehicles dominate the dataset.
* Petrol vehicles represent the largest market share.
* Individual sellers contribute significantly to listings.
* Vehicle age has a strong impact on selling price.
* Older vehicles experience higher depreciation.
